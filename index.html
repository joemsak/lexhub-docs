Lexhub

Find out your most commonly used words in commits, I guess.

Documentation

Installation

Add this line to your application's Gemfile:

gem 'lexhub'

And then execute:

$ bundle

Or install it yourself as:

$ gem install lexhub

Usage

You will need a Github Application registered with an auth token.
